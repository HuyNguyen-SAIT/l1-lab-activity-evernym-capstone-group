
-----------------------
HOW TO INSTALL DATABASE
-----------------------
Step1. Go to Services tab
Step2. Click right on Databases-MySQL Server-Create Database
Step3. Name the Database
Step4. After the new icon is created at the bottom of Databases section,
       right click on it and then click Execute Command...
Step5. Run NotesDB.sql file or scripts for creating the database.
